# Лабораторная работа №1: Первичное исследование и оценка качества данных

## Описание проекта
Данный проект выполнен в рамках учебного курса «Интеллектуальный анализ данных».Основная цель работы — проведение первичного аудита качества данных (Data Audit) на примере набора данных о продажах видеоигр, выявление типичных проблем и визуализация ключевых закономерностей.

## Исходные данные
Для анализа используется табличный датасет `vgsales.csv`, содержащий более 16,500 записей о продажах видеоигр.В наборе данных представлены как числовые (объемы продаж), так и категориальные признаки (жанры, платформы, издатели).
## Структура проекта
* `data/` — содержит исходный файл с данными `vgsales.csv`.
* `notebooks/` — содержит Jupyter Notebook `01_data_understanding.ipynb` с процессом исследования.
* `report/` — содержит текстовый отчёт о качестве данных `quality_report.md`.
* `README.md` — краткое описание проекта и инструкции по запуску.

## Инструкции по запуску
Для воспроизведения результатов исследования выполните следующие шаги:

1. **Клонируйте репозиторий** (или скачайте файлы проекта).
2. **Установите необходимые зависимости**:
   Для работы требуются библиотеки `pandas`, `seaborn` и `matplotlib`. Установить их можно командой:
   ```bash
   pip install pandas seaborn matplotlib
   
3. **Откройте проект в VS Code:** Убедитесь, что установлены расширения Python и Jupyter.

4. **Запустите ноутбук:** Откройте файл notebooks/01_data_understanding.ipynb и выполните все ячейки по порядку («Run All»). Код должен выполняться «с нуля» без ошибок.
