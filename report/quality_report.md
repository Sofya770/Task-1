# Отчёт о качестве данных: Исследование продаж видеоигр

## 1. Введение
Данный отчёт посвящён первичному анализу датасета о продажах скоропортящихся товаров в розничной сети. Набор данных содержит информацию о 440 транзакциях по различным категориям товаров (продукты питания, фармацевтика, готовая еда) с детализацией по условиям хранения, срокам годности, продажам и убыткам от порчи.

Условным заказчиком анализа выступает отдел логистики и управления запасами розничной сети, заинтересованный в оптимизации процессов управления скоропортящимися товарами. В ходе работы решены задачи поиска аномалий (выявление паттернов убытков) и описательной аналитики (профилирование категорий товаров по уровню риска).
---

## 2. Паспорт датасета
Объект исследования представляет собой таблицу со следующими характеристиками:

**Размер:** 440 строк и 46 столбцов.
**Ключевые признаки:** 

| Признак | Описание | Тип данных |
| :--- | :--- | :--- |
| `product_id` | Уникальный идентификатор товара | Категориальный |
| `product_name` | Название товара | Категориальный |
| `category` | Категория товара | Категориальный |
| `transaction_date` | Дата продажи | Временной |
| `profit` | Прибыль от транзакции | Числовой |

---

## 3. Основные проблемы качества данных
В результате проведённого аудита было выявлено 5 ключевых проблем:

| № | Тип проблемы | Описание | Риск для анализа | Стратегия решения |
| :--- | :--- | :--- | :--- | :--- |
| 1 | Отрицательные значения | Отрицательные значения в ценах, количестве и температуре | Искажение статистических показателей | Проверка на корректность данных, возможно, преобразование |
| 2 | Несогласованность | Отрицательная температура хранения для некоторых категорий | Некорректный анализ условий хранения | Проверка единиц измерения и логики данных |
| 3 | Выбросы | Аномально высокая и низкая прибыль в profit | Смещение средних значений | Использование медианы для выводов |
| 4 | Подозрения | Процент утилизации (waste_pct) > 100% | Невозможность интерпретации | Ограничение диапазона или проверка формулы расчета |

---

## 4. План улучшения качества
Для подготовки данных к дальнейшим этапам интеллектуального анализа предлагаются следующие шаги:

1. **Очистка (Приоритет: Высокий):** Проверка и коррекция отрицательных значений в ключевых числовых полях (цена, количество, температура). Определение, являются ли они ошибками или допустимыми значениями.
2. **Трансформация (Приоритет: Средний):** Приведение всех временных меток к единому формату datetime для корректного анализа временных рядов и сроков годности.
3. **Обработка выбросов (Приоритет: Низкий):** Для анализа прибыльности и убытков применить методы обнаружения выбросов (IQR, z-score) с последующим решением об исключении или трансформации.

---

## 5. Краткое резюме
В ходе выполнения работы были освоены навыки первичного исследования данных с помощью библиотек `pandas` и `seaborn`. Основное внимание при дальнейшем анализе следует уделить факторам, влияющим на утилизацию товаров, и оптимизации условий хранения для различных категорий продукции. Ключевой задачей является снижение финансовых потерь от порчи при поддержании необходимого уровня доступности товаров для покупателей.
